HTML5之元素与标签结构

2016/8/31 posted in  HTML5  

一 . HTML5语法

1. DOCTYPE及字符编码

<!DOCTYPE html>
<html lang="zh-CN">
    <head>
        <meta charset="utf-8">
    </head>
</html>

　　1⃣️ 字符集 简化为 <!DOCTYPE html>
　　2⃣️ 给文档指定语言 < html lang="zh-CN">
　　3⃣️ 字符编码 < meta charset="utf-8">

　　定义性语言会变的简单。　　

2. 大小写都可以

<Input tYpe=Radio/>

　　这种大小写混用的方式是可以的，但是极其不推荐这样写，在 HTML4中也是可以，但是XHTML是不可以大小写混写的。

3. 布尔值

HTML4:需要重写一遍
　　<input type="checkbox" checked="check"/>
HTML5:不需要重写,默认不写时为false。
　　<input type="checkbox" checked/> 为true

4. 省略引号

　　属性的双引号可以写成单引号，也可以省略不写，但这两个都不推荐，推荐双引号。

5. 可以进行省略的标签

　　< input> < /input>这一类标签不允许写结束标签，只能写< input />
　　
　　可以省略结束符的和可以完全省略的都不建议使用。
　

二 . 新增及删除的标签

1 . 新增标签

1⃣️ 结构标签

< section>标签 　
　　< section> 标签定义文档中的节（section、区段）。比如章节、页眉、页脚或文档中的其他部分，可以和其他标签如h1,h2..进行组合。
　　　　
< article>标签
　　表示页面中与上下文不相关的独立内容。比如一篇文章。
　　
< aside>标签
　　表示article内容之外，与article标签内容相辅的信息

< header>标签
　　表示页面一个区域快或整个页面的标题
　　
< hgroup>标签
　　表示对整个页面或一个内容块的标题进行组合。
　　　
< footer>标签
　　一个页面的脚注。
　　
< nav>标签
　 表示页面中的链接导航部分。

< figure>标签
　 标签规定独立的流内容（图像、图表、照片、代码等等）。

2⃣️ 表单标签

  • email 　 必须输入email
  • url 　 必须输入url地址
  • number 　 必须输入数值
  • range 　 必须输入一定范围内数值
  • Data Pickers(日期选择器)
  • search 　 用于搜索域，域显示为常规的文本域

3⃣️ 媒体标签

< video>标签 定义视频。
HTML4:

<object type="video/ogg" data="move.opv">
    <param name="src" value="movie.ogv" />
</object>

HTML5:

<video src="moive.ogg" controls="controls">video标签</controls>

< audio>标签 定义音频
HTML4:

<object type="application/ogg" data="someaudio.wav">
        <param name="src" value="someaudio.wav" />
</object>

HTML5:

<audio src="someaudio.wav">audio标签</audio>

< embed>标签
用来嵌入内容（各种媒体）。

HTML4:

<object data="flash.swf" type="application/x-shockwave-flash"></object>

HTML5:

<embed src="flash.swf">

4⃣️. 其他功能标签

mark 标签
代码：

<p> I AM A <mark>BOY</mark>

例子：

I AM A BOY

progress 标签
代码：

<progress max="100" value="85"><span>85</span>%</progress>

例子：

85%

detail标签

代码：

<details open>
   <summary>Macbook pro</summary>
   <ul>
     <li>iphone1</li>
     <li>iphone2</li>
     <li>iphone3</li>
   </ul>
</details>

例子：


Macbook pro

  • iphone1

  • iphone2

  • iphone3

1⃣️.可以用css代替的标签

2⃣️.不再使用frame

3⃣️.只有个别浏览器支持的标签

4⃣️.其他不常用的标签
